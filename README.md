 Instructions for use
 1. open vs studio code
 2. install following packages: 
 - Rust by: 1YiB extensions for rust
 - Extension Pack (3)
     - rust-analyzer
     - Rust language support for Visual Studio Code
     - The Rust Programming Language
 3. create a new enviroment in vs code
 4. Copy main file into main.rs
 5. replace cargo.toml with the one found in repo

 6. In bash, cd <foldername>
mine is cd rust-lab as a example
7. next type cargo run
8. enjoy!!!
